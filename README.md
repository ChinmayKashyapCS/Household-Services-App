# 🛠️ Household Services App

A full-stack application that connects users with household service providers to request, schedule, and manage services such as plumbing, cleaning, and repairs.

---

## 📖 Overview

The Household Services App is designed to simplify booking and managing home services by matching users with available service providers. Users can browse services, place requests, view status, and track service history. Providers can manage service requests and update their availability.

---

## 🧠 Key Features

- User registration and authentication
- Service listing and browsing
- Booking and scheduling services
- Provider request management
- View service history and status updates
